## 🚀 项目简介

**Tomato Server** 是一个基于 **Boost.Asio** 的现代 C++17 TCP 服务器框架，特点：

- 异步网络通信，使用 Boost.Asio
- 逻辑线程池处理业务逻辑
- MySQL 数据库线程池异步查询
- 支持多客户端并发请求
- 高性能、模块化设计，易扩展
